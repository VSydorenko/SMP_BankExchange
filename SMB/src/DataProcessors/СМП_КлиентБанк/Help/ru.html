Обмен с банком

Обработка позволяет организовать передачу в банк платежных поручений и получение выписок путем прямого подключения к сервису банка или через специализированную программу типа "Клиент-Банк".

Настройка обмена с банком

Настройка производится из формы обработки для каждого расчетного счета организаций предприятия. Если в организации открыто несколько расчетных счетов, то необходимо сделать несколько настроек. При выборе банковского счета будет автоматически установлена нужная настройка.

Для прямого обмена данными с банком потребуется создание Настройки ЭДО для обмена с банком и перевод его в статус Действует.

Поддерживаемые сценарии

Обработка Обмен с банком позволяет:

  • отправлять в банк по электронным каналам связи платежные поручения, сформированные в информационной базе;
  • загружать в базу информацию об операциях по расчетным счетам организации из банка.

В зависимости от настроенного варианта обмена с банком, отправка платежных поручений в банк и получение выписок из банка может производиться напрямую из банка, без использования дополнительных программ, или через программу типа "Клиент-Банк". Если для банка, указанного в выбранном счете действует настройка прямого обмена, будет использоваться прямой обмен. Если настройка прямого обмена не задана или счет не указан, будет использоваться обмен через программу типа "Клиент-Банк".

Получить наглядное представление о работе прямого обмена с банком поможет видеоролик http://www.youtube.com/watch?v=nUolZlb_EKQ

 

Обмен данными между конфигурацией и программой типа "Клиент-Банк" осуществляется через промежуточное хранилище информации, в качестве которого используется файл обмена специального формата.

Для отправки платежных документов подготовленные в конфигурации данные выгружаются в файл обмена. Программа типа "Клиент банка" этот файл читает и обрабатывает. Результат обработки передается по телекоммуникационным каналам в банк.

Для получения банковских выписок программой типа "Клиент-Банк" инициируется прием данных из банка и формирование файла обмена. Затем этот файл читается и обрабатывается.

Необходимое условие для взаимодействия с программой типа "Клиент-Банк": банковская программа должна поддерживать стандарт обмена данными "1С:Предприятие — Клиент-Банк".
